# Интернет‑магазин мебели (e‑commerce)

*Учебный проект по frontend‑разработке*


## Описание проекта

Интернет‑магазин мебели — учебный проект, созданный для отработки и демонстрации навыков современной frontend‑разработки.

Сайт включает основные страницы:
* **Главная** — презентация ассортимента и акций;
* **Каталог** — фильтрация и сортировка товаров;
* **Корзина** — оформление заказа.

**Цель проекта:** отработать навыки вёрстки и создания интерактивного пользовательского интерфейса.

**Бизнес‑цель сайта:** повысить продажи за счёт информативного интерфейса, удобного процесса оформления заказа и улучшенного пользовательского опыта.

## Особенности проекта

* **Адаптивный дизайн.** Сайт корректно отображается на мобильных устройствах, планшетах и десктопах.
* **Интерактивные элементы.** Реализованы эффекты наведения и плавные переходы для улучшения UX.
* **Формы взаимодействия.** Встроены формы подписки на рассылку и оформления заказа.
* **SEO‑оптимизация.** Семантическая вёрстка, корректная иерархия заголовков и мета‑теги для лучшей индексации поисковыми системами.
* **Оптимизация производительности.** Ускоренная загрузка за счёт:
  * использования формата WebP для изображений;
  * встраивания SVG‑иконок;
  * оптимизации загрузки веб‑шрифтов (Google Fonts).
* **Доступность (Accessibility).** Сайт адаптирован для пользователей с ограниченными возможностями:
  * альтернативный текст (`alt`) для всех изображений;
  * логическая структура заголовков;
  * совместимость со скринридерами.

## Технологии

* **HTML5** — семантическая разметка;
* **CSS3** — стилизация и визуальные эффекты:
  * Flexbox и Grid для вёрстки;
  * анимации и переходы;
  * адаптивные медиазапросы;
* **SVG** — векторные иконки и графика;
* **WebP** — оптимизированные изображения для быстрой загрузки;
* **Google Fonts** — подключение веб‑шрифтов;
* **Git и GitHub** — контроль версий и деплой.

## Запуск проекта локально

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/kors-ak/site.git

2. Перейдите в папку проекта:
   ```bash
   cd site

3. Откройте файл index.html в браузере или запустите локальный сервер (например, с помощью Live Server в VS Code).

## Правовая информация

Проект создан в учебных целях. Не предназначен для коммерческого использования. Все материалы (текст, изображения, дизайн) используются исключительно для демонстрации навыков разработки. Текст на страницах является вымышленным и служит примером наполнения интернет‑магазина.
