# Интернет-магазин мебели (e-commerce)
*Учебный проект по frontend-разработке*

### Описание проекта: 
Сайт‑каталог мебели с возможностью оформления заказа.  
**Основная задача сайта** — повышение продаж через информативный интерфейс, обеспечивающий удобство оформления заказа и улучшение пользовательского опыта.

### Особенности проекта:

**Адаптивность.** Сайт корректно отображается на мобильных устройствах, планшетах и десктопах.  
**Интерактивные элементы.** Реализованы эффекты наведения и плавные переходы.  
**Формы подписки и заказа.** Встроены формы для подписки на рассылки и для оформления заказа.  
**SEO-оптимизация.** Проект оптимизирован для поисковых систем (семантическая вёрстка).  
**Оптимизация загрузки.** Ускоренная загрузка шрифтов, использование WebP‑изображений, встроенные SVG.  
**Доступность (Accessibility).** Сайт адаптирован для скринридеров (альтернативный текст для изображений и логическая структура заголовков).


### Использованные технологии:

HTML5;  
CSS3 (Flexbox, Grid, анимации, адаптивные медиазапросы);  
SVG (векторные иконки);  
WebP (оптимизированные изображения);  
Google Fonts (подключение веб‑шрифтов);  
Git и GitHub для контроля версий и деплоя.
